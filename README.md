# Morse Code Translator

A simple web application that converts text to Morse code and Morse code back to text.

## Features

* Text → Morse translation
* Morse → Text translation
* Simple and user-friendly interface
* Modern and responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```
project-folder
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## Installation and Running

1. Download or clone the project:

```
git clone <repo-link>
```

2. Open the project folder and run the `index.html` file in your browser.

## Usage

1. Enter text or Morse code in the input field.
2. Click the **Translate** button.
3. The result will appear on the screen.

## Morse Code Format

* Letters are separated by spaces
* Words are separated by `/`

Example:

```
HELLO → .... . .-.. .-.. ---
.... . .-.. .-.. --- → HELLO
```
